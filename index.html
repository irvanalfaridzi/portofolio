Project

UKMKU

UKMKu is an application that helps students at Dinamika University in choosing UKM fields according to their talent interests. UKMKU uses the RIASEC method in the process of detecting student interests and talents. To be able to provide UKM recommendations to students, the SAW method is used by using the 3 criteria for talent interest rankings and other criteria to produce a ranking in the UKM sector.

The UKMKU website was created using the Laravel framework. It helps Bagian Kemahasiswaan to manage criteria from UKMKU apps.

UKMKU app was created using the Flutter Framework. This application is used by students in Dinamika University to find out the fields of UKM that match their interests, talents, and criteria that have been predetermined by the Bagian Kemahasiswaan.
